## Como adicionar novos contribuidores

1. Confirme com a pessoa que ela concorda em aparecer nesta lista e em qual função/área deseja ser identificada.
2. Abra um PR atualizando `docs/CONTRIBUTORS.mdx`, adicionando-a na seção adequada (core ou convidados) com uma breve descrição das responsabilidades.
3. Referencie no PR qualquer issue ou documento que descreva o escopo de atuação da pessoa.
4. Solicite revisão dos mantenedores antes do merge.

Este documento é revisado manualmente pelos mantenedores e faz parte do checklist de onboarding.
